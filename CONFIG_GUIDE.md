# 챗봇 설정 가이드 (Configuration Guide)

이 문서는 챗봇을 운영하고 새로운 챗봇을 추가할 때 수정해야 하는 설정 파일들에 대한 설명입니다.

## 1. 🤖 챗봇 설정 (`lib/bots.ts`)
가장 중요한 설정 파일입니다. 챗봇의 이름, 성격(프롬프트), 초기 메시지 등을 여기서 설정합니다.

### 주요 설정 항목
- **`id`**: 챗봇의 고유 ID입니다. URL(`.../chat/아이디`)과 데이터 폴더명(`data/아이디`)과 일치해야 합니다. (영문 소문자 권장)
- **`chatbotName`**: 채팅창 상단에 표시될 이름입니다.
- **`systemPrompt`**: 챗봇의 페르소나와 규칙을 정의하는 핵심 프롬프트입니다.
- **`initialMessages`**: 채팅방에 처음 들어갔을 때 표시되는 환영 메시지입니다.
- **`rag.coreFiles`**: 항상 참조해야 할 핵심 파일 목록입니다. (예: `['한의학역사.md']`)
- **`googleSheetId`**: (선택사항) 대화 로그를 저장할 구글 스프레드시트 ID입니다. 비워두면 기본 시트에 저장됩니다.

```typescript
// 예시
'oriental': {
    id: 'oriental',
    chatbotName: "한의학 AI 컨설턴트",
    systemPrompt: "...",
    // ...
}
```

## 2. 🔑 비밀키 설정 (`.env`)
API 키와 같은 보안 정보는 이 파일에서 관리합니다. **이 파일은 절대 깃허브에 올리지 마세요.**

### 필수 항목
- **`GOOGLE_GENERATIVE_AI_API_KEY`**: Google Gemini AI를 사용하기 위한 API 키입니다.
- **`GOOGLE_SHEETS_...`**: 구글 스프레드시트 연동을 위한 인증 정보들입니다. (로그 저장용)

## 3. 📚 데이터 폴더 (`data/`)
설정 파일은 아니지만, 챗봇의 지식이 되는 파일들이 위치하는 곳입니다.
- **`data/[botId]/raw/`**: 원본 PDF 파일을 넣는 곳입니다.
- **`data/[botId]/`**: 변환된 MD 파일과 `knowledge_index.json`이 위치하는 곳입니다.

## 4. 📝 기타 설정
- **`next.config.ts`**: 웹사이트 자체의 기술적인 설정(이미지 도메인 허용 등)이 필요할 때 수정합니다. (보통은 건드릴 필요 없음)
