## 4. 실행 방법 (How to Run)

### 개발 서버 실행
```bash
npm run dev
```

### 새로운 한의학 자료 추가
1. `data/raw` 폴더에 PDF 파일 넣기.
2. 변환 스크립트 실행: `npm run convert-pdf`
3. `lib/config.ts`에 파일 등록.
