# PDF 변환 및 데이터 추가 가이드

이 가이드는 PDF 파일을 챗봇이 학습할 수 있는 마크다운(.md) 형식으로 변환하고, 설정에 추가하는 방법을 설명합니다.

## 1. PDF 파일 준비
변환할 PDF 파일을 아래 폴더에 넣어주세요:
`c:\chatbot\data\raw`

## 2. 변환 명령어 실행
터미널에서 아래 명령어를 복사하여 실행하세요:

```bash
npm run convert-pdf
```

이 명령어를 실행하면 `data/raw` 폴더에 있는 모든 PDF 파일이 텍스트로 추출되어 `data/` 폴더에 같은 이름의 `.md` 파일로 저장됩니다.

## 3. 설정 파일 업데이트
변환된 파일이 챗봇의 지식베이스에 포함되도록 `lib/config.ts` 파일을 수정해야 합니다.

1. `c:\chatbot\lib\config.ts` 파일을 엽니다.
2. `rag.coreFiles` 배열에 새로운 파일명을 추가하거나, `rag.keywordMap`에 키워드와 함께 매핑합니다.

### 예시:
만약 `자연요법가이드.pdf`를 변환했다면 `자연요법가이드.md`가 생성됩니다.

**모든 질문에 대해 항상 참조하게 하려면:**
```typescript
coreFiles: ['knowledge_base.md', '자연요법가이드.md'],
```

**특정 키워드(예: "요법")가 있을 때만 참조하게 하려면:**
```typescript
keywordMap: {
    '요법': ['자연요법가이드'],
    // ...
}
```
